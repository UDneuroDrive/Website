    neuroDrive

    neuroDrive will be a fully autonomous small scale vehicle that utilizes computer vision and deep learning to learn and navigate through an imposed environment.

    The neuroDrive Project

    1. GitHub Repo

    2. Design: For this project we will be designing a fully autonomous small scale vehicle that utilizes computer vision and deep learning to learn and navigate through an imposed environment. This project will emulate the challenges auto makers are facing but on a smaller scale. Using this technology the vehicle will be able to navigate with intention through an environment we set up.

    3. Mission : We believe using a small scale, easy to control and easy to manipulate environment, we can do testing that could potentially be useful in the production of full size autonomous vehicles.

    4. Success: This project has three main goals and would be considered a success if we meet all three of them. The first of these goals is to create an autonomous rc car that is able to stay within its lane on both straight and curved roads. The second goal is for the autonomous rc car to obey basic traffic signals/signs such as stop signs, speed limit signs, stop lights etc. Our third and final goal is for the rc car to be able to detect and avoid both stationary and moving obstacles. Ideally, it would be able to analyze the threats and chose the path that would cause the least amount of injury to itself and other people.

    5. Customer: Our target customers are companies that are developing self driving cars so they can test out decision making during ai development. This will allow for much easier testing by shrinking the required space to run tests. Instead of needing a space the size of multiple city blocks to test a full sized autonomous vehicle, they will be able to test on nearly any open space. These small scale tests will also make it easier to control because the environment is smaller.
